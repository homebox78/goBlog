# Blogger 페이지 초안 (애드센스 준비용)

'생활꿀통365' Blogger 블로그에 붙여넣을 정책/소개 페이지 초안입니다.
애드센스 승인 심사에서 소개·개인정보처리방침·문의 경로는 사실상 필수입니다.

## 붙여넣는 법
1. Blogger 관리자 → **페이지** → **새 페이지**
2. 우측 상단 편집기를 **HTML 보기**로 전환
3. 아래 파일 내용을 그대로 붙여넣기 → 제목 입력 → **게시**
4. 게시 후 **레이아웃 → 페이지 가젯**(또는 하단 링크 목록)에 세 페이지를 연결해 모든 글에서 접근 가능하게

| 파일 | 페이지 제목 | 용도 |
|---|---|---|
| `소개.html` | 소개 | 블로그 정체성 + 제휴 고지 |
| `개인정보처리방침.html` | 개인정보처리방침 | 애드센스 필수 (쿠키·제3자 광고 고지) |

## 확인 후 바꿀 값
- 이메일 주소(`homebox000001@gmail.com`)를 실제 문의 받을 주소로 확인
- 시행일(2026-07-11)은 실제 게시일로 조정 가능
- 블로그 이름('생활꿀통365')이 실제와 다르면 일괄 수정

문의 페이지는 별도 페이지 대신 소개/개인정보처리방침 하단의 이메일로 갈음할 수 있으나,
심사 안정성을 위해 이메일을 명시한 '문의' 페이지를 따로 두는 것도 권장됩니다.
